2D Embedded Racing Video Game Project 

AUTHOR: Aimee Darlene Izere

TECH: LANGUAGE: ARM Assembly Programming, PROGRAMMING SOFTWARE: CPUlator, ARMv7 Architecture, Nios II DE1 SoC System 

GAME DESCRIPTION: Move right, left, up or down to avoid cars and collect a maximum of 3 power-ups to survive longer

3 ADVANCED FEATURES:
						1. Gravity or Other Forces : pushing buttons 1 and 2 simultaneously allows the player to jump over cars
						2. PowerUps: A player can get a max of 3 powerups throughout the game
						3. Complex Scoring Sytem with catching powerups to survive crashes. 
						 If you crash into a car and you have > 0 power-ups, you survive that crash but you lose 1 power up
						  I also Display to the screen the number of powerUps that a player has remaining

HOW TO PLAY THE GAME: Use CPUlator Push buttons: * Button 0 : Move right				
											* Button 1 : Move Up
											* Button 2 : Move Down
											* Button 3 : Move left
											* Button 1 + Button 2 simultaneously : Jump
											NOTE : To stop the 5player from moving, release the push buttons
